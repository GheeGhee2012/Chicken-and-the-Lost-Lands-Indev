To set up the game follow the following instructions. you will only need to do this when you first download it(the windows set up is going to be easier in a future update!)

LINUX:

1: Download python3 at python.org

2: run launch.sh to install requirements
other than python (download at python.org)

3: Launch catll.pyw!

WINDOWS:
1:download python3 at python.org

2:goto cmd and write 'pip install pysimpleGUI==4.60.1'

3:rename main.py to catll.pyw

4:launch catll.pyw

5:delete launch.sh
